Case 2: Graph Traversal.

This page contains an implementation of Breadth First Search(BFS) for graph traversal. To test it we test it on a number of graphs. Unfortunately for one case it doesn't work correctly. To run the program, open firebug and select console panel. Once you click on the button (Traverse Graphs!), program runs BFS algorithm on every given graph and error message appears. Locate the defect cause this error.
