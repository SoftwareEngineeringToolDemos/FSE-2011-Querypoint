lastChange User Study

Thank you for participating in this user study. The goal of this user study is to assess the usability of a new functionality, lastChange, in Firebug.

Through this user study which takes about 45 minutes you must to debug two buggy programs and report the defects.

Please go through the following setup steps:

  • Make sure you have installed Firefox 3.6.X on your computer.
  • Create a new Firefox profile dedicated for this user study and open the Firefox with this profile.
  • Install Firebug 1.7 .
  • Install Querypoint extension.
  • Close and re-open Firefox.
