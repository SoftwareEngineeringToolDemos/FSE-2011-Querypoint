This page contains an object "myObject", whenever user clicks, "myObject.myProperty" value increases by 3.

To see how lastChange works set a breakpoint on line 17-where the value is written on the page. Reload the page and click. The breakpoint hits. In the right panel, check the value of this.myObject.myProperty. The value is 3. Now right click on myProperty and select lastChange. As you see, in QP panel you can see the location of the last change of myObject.myProperty and the program state at the right side. You can still ask for another lastChange. From the right panel select window.myObject.myProperty. Right click on myProperty and select lastChange. Now debugger shows a new location. All three locations, breakpoint, the first last change and the second last change are listed in the menu at the top of the panel.

myObject.myProperty = 0;
